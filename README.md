# 💻 Teknoloji Kullanımı ve Psikolojik Etki Analizi

📌 Proje Özeti: Bu çalışma, günlük teknoloji kullanım alışkanlıklarının (ekran süresi, uygulama tercihleri vb.) bireylerin psikolojik iyi oluş halleri üzerindeki etkilerini analiz etmek amacıyla geliştirilmiştir.

🛠️ Teknik Analiz Süreci:

Keşifsel Veri Analizi (EDA): Veri setindeki genel eğilimlerin belirlenmesi ve temel istatistiksel metriklerin (ortalama, standart sapma vb.) hesaplanması.

Korelasyon Analizi: Farklı teknoloji kullanım türleri ile ruh sağlığı skorları arasındaki ilişkinin ölçülmesi.

Görselleştirme: Analiz sonuçlarının Python (Matplotlib/Seaborn) kütüphaneleri kullanılarak grafiklere dökülmesi.

🚀 Kullanılan Teknolojiler:

Dil: Python

Kütüphaneler: Pandas, NumPy, Seaborn, Matplotlib
